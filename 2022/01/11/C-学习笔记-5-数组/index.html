概述

所谓数组，就是一个集合，里面存放了相同类型的数据元素。

特点：

  • 数组中的每个数据元素都是相同的数据类型
  • 数组是由连续的内存位置组成的

一维数组

定义方式

一维数组定义的三种方式：

  1. 数据类型 数组名[ 数组长度 ];
  2. 数据类型 数组名[ 数组长度 ] = { 值1, 值2... };
  3. 数据类型 数组名[ ] = { 值1, 值2... };

数组名

一维数组名称的用途：

  1. 可以统计整个数组在内存中的长度
  2. 可以获取数组在内存中的首地址

案例：五只小猪称体重。在一个数组中记录了五只小猪的体重，找出并打印最重的小猪体重。

案例：数组元素逆置。声明一个含有5个元素的数组，并且将元素逆置。

冒泡排序

**作用：**最常用的排序算法，对数组内的元素进行排序

  1. 比较相邻的元素，如果第一个比第二个大，就交换他们两个；
  2. 对每一个相邻的元素做同样的操作，执行完毕后，找到第一个最大值；
  3. 重复以上步骤，每次比较次数 -1，直到不需要比较为止。

二维数组

二维数组就是在一维数组之上，多加一个维度。

定义方式

二维数组一般有四种定义方式：

  1. 数据类型 数组名[ 行数 ][ 列数 ];
  2. 数据类型 数组名[ 行数 ][ 列数 ] = { {数据1, 数据2}, {数据3, 数据4}}；(推荐使用，提高代码的可读性)
  3. 数据类型 数组名[ 行数 ][ 列数 ] = { 数据1, 数据2, 数据3, 数据4};
  4. 数据类型 数组名[ ][ 列数 ] = { 数据1, 数据2, 数据3, 数据4};
